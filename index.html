Изучение web технологий

HTML (от английского HyperText Markup Language) — это язык гипертекстовой разметки текста. Он нужен, чтобы размещать на веб-странице элементы: текст, картинки, таблицы и видео. Парные теги применяются для разметки блоков — элементов, в которые можно вкладывать другие элементы (в том числе блоки).

К ним относятся:

  • контейнеры (div)
  • абзацы
  • заголовки
  • списки
  • таблицы
